# 个人 Ops — AI 驱动的个人生活管理

> 本文档为英文版的中译本。如有歧义，以英文原版为准。

---

## 功能概述

个人 Ops 将你的待办清单变成一个**AI 驱动的每日驾驶舱**。你不需要手动整理生活，只需把所有东西扔到收件箱，AI 每天早上为你生成一份聚焦简报。

---

## 核心概念

### 收件箱（SSoT）

`00-INBOX/todo-backlog.md` 是你所有待办事项的**唯一真相源**。

一切首先进入这里。没有例外。

### 优先级体系

| 级别 | 含义 | 行动 |
|------|------|------|
| **P0** | 今天必须完成 | 出现在今日 Top 3 |
| **P1** | 本周推进 | 排入周计划 |
| **P2** | 本月安排 | 加入月里程碑 |
| **P3** | 需要判断 | 留在待办池直到评估 |

### 每日简报

AI 每天早上 07:00 在 `01-PERSONAL-OPS/01-DAILY-BRIEFS/daily-briefing.md` 生成简报。

结构：
```
一、今天最重要的 3 件事
二、今天必须推进但不必做完
三、今天等待反馈 / 需要催办
四、今天需要拍板的事
五、今天可委派的事
六、低能量时可做的小事
七、今天明确不做
八、今日提醒
```

### 决策规则

AI 生成简报时遵循以下规则：
1. 截止日期为今天/明天的优先
2. 高杠杆但未启动的次之
3. 已完成/已关闭的绝不作为活跃项呈现
4. 用户的条目不会被删除（只会被重排序、合并、降级）

## 每日循环

```
07:00  ┌─ Morning Brief ──────────────────────┐
       │  AI 读取: todo-backlog、承诺、决策、      │
       │  进度看板                                   │
       │  生成: daily-briefing.md                   │
       └────────────────────────────────────────┘
       │
 全天   你按照简报推进工作
       │
15:00  ┌─ Afternoon Reminder ──────────────────┐
       │  "检查今天的 P0 进度"                        │
       └────────────────────────────────────────┘
       │
20:00  ┌─ Evening Review ─────────────────────┐
       │  "有更新吗？哪些延续了？"                    │
       └────────────────────────────────────────┘
       │
 随时   新条目 → todo-backlog.md（P0-P3）
       │
 周一   生成周计划
 每月1日 检查月里程碑
```

## 添加新条目

### 快速添加

直接追加到 `00-INBOX/todo-backlog.md`：

```markdown
- [ ] [P1] Call the dentist
  - 领域：健康
  - 重要性：中
  - 紧急性：高
  - 下一动作：Call Dr. Smith at 555-0123
  - 截止：2026-04-10
```

AI 会在下一次早间简报中处理它。

### 批量导入

如果你有一堆想法，直接全部列在待办池里。AI 会在下一次简报生成时进行分诊。

## AI 能做什么和不能做什么

### ✅ AI 能
- 按优先级和截止日期重排序
- 合并重复项
- 降级停滞项
- 生成结构化简报
- 跟踪后续事项和提醒
- 识别决策瓶颈

### ❌ AI 不能
- 删除你的条目（只有你能关闭/归档）
- 为你做约束性决策
- 在日历上安排会议
- 联系外部人员

## 自定义

### 更改简报时间

编辑 cron job：
```json
{
  "schedule": {
    "expr": "0 7 * * *",  // 改为你偏好的小时
    "tz": "Your/Timezone"
  }
}
```

### 添加简报小节

编辑 `prompts/personal-ops-morning-brief.prompt.md` 以添加自定义小节。

### 禁用提醒

在 cron 配置中设置 `personal-ops-todo-reminder-1500` 和 `personal-ops-todo-reminder-2000` 的 `"enabled": false`。
